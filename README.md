Task 2: Data Visualization and Storytelling

Objective

To create interactive and meaningful visualizations that clearly communicate business insights and trends from sales data using Power BI.

Dataset Used

Superstore Sales Dataset (Sales data containing information about orders, sales, profit, categories, regions, and dates).

Steps Performed

• Imported the Superstore dataset into Power BI Desktop

• Cleaned and verified data types for sales, profit, and date fields

• Created KPI cards to highlight Total Sales, Total Profit, and Total Orders

• Built a bar chart to analyze Total Sales by Product Category

• Created a column chart to visualize Profit Performance by Sub-Category

• Designed a line chart to show Sales Trend Over Time (Year-wise)

• Developed a region-wise bar chart to compare Sales across different regions

• Added slicers for Region and Category to enable interactive filtering

• Applied proper formatting, labels, and minimal colors to improve readability

• Added text-based insights below each visual to support data storytelling

• Organized all visuals into a clean and professional dashboard layout

Tools Used

• Power BI Desktop

Output

• Interactive Power BI Dashboard
